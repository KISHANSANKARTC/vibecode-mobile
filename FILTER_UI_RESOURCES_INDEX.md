## 🎨 Key Design Values to Remember

### Spacing
- Icon: 36x36px (sections) or 32x32px (rows)
- Padding: 16px inside cards
- Gap: 20px between sections
- Icon-text gap: 12px

### Colors
- Gray (unselected): #f3f4f6 background, #6b7280 text
- Purple (selected): #7c3aed background, #ffffff text
- Icons: #6b7280 (always gray)

### Sizing
- Minimum touch: 44x44px
- Toggle row: 56px height
- Border radius (cards): 16px
- Border radius (chips): 8px

---

## 📊 Issue Priority (Fix in This Order)

1. **Text wrapping** (countries, long text)
2. **Icon misalignment** (not vertical center)
3. **Button text** (not centered)
4. **Spacing** (inconsistent padding/gaps)
5. **Touch targets** (smaller than 44px)
6. **Colors** (inconsistent selected state)
7. **Polish** (borders, radius, shadows)
